````markdown
# 电影推荐项目第二步：经典推荐算法学习、实现与对比实验

本阶段对应课程任务中的第二部分，目标是学习并实现经典推荐算法，完成至少两类算法的实验对比，并对模型参数进行系统分析。

当前阶段优先实现以下三类方法：

1. 基线方法
   - 全局平均分
   - 用户平均分
   - 电影平均分

2. 协同过滤方法
   - 基于物品的协同过滤（ItemCF）

3. 矩阵分解方法
   - 带偏置项的矩阵分解（BiasMF）

其中，协同过滤和矩阵分解属于两种不同类型的推荐算法，已经满足“至少 2 种类型算法对比”的要求。图神经网络模型可作为后续扩展方向继续补充。
````

## 五、本阶段完成的主要内容

### 1. 数据划分

将评分数据划分为：

* 训练集
* 验证集
* 测试集

默认划分比例为：

* 训练集：0.8
* 验证集：0.1
* 测试集：0.1

并固定随机种子，保证实验可复现。

### 2. 基线模型

实现以下基线方法：

* 全局平均分
* 用户平均分
* 电影平均分

这些方法作为后续协同过滤和矩阵分解模型的对照基准。

### 3. 协同过滤模型

实现基于物品的协同过滤方法，支持：

* 邻居数参数 `k`
* 相似度类型选择

  * 余弦相似度
  * 皮尔逊相关系数
* 最少共同评分数阈值

### 4. 矩阵分解模型

实现带偏置项的矩阵分解模型，预测公式为：

```text
预测评分 = 全局平均分 + 用户偏置 + 电影偏置 + 用户隐向量与电影隐向量内积
```

支持以下参数：

* 隐向量维度
* 学习率
* 正则化系数
* 训练轮数

### 5. 模型评价指标

统一使用以下指标评价模型：

* MAE
* RMSE

并分别在验证集和测试集上输出结果。

### 6. 参数分析

本阶段支持对以下参数进行系统实验分析：

#### ItemCF 参数分析

* `k`
* `sim_metric`
* `min_common`

#### BiasMF 参数分析

* `n_factors`
* `lr`
* `reg`
* `epochs`

## 六、输出结果说明

```text

├── baseline_results.csv
├── itemcf_results.csv
├── mf_results.csv
└── summary.json
```

各文件说明如下：

### 1. `baseline_results.csv`

保存基线模型在验证集和测试集上的结果，包括：

* 模型名称
* 验证集 MAE / RMSE
* 测试集 MAE / RMSE

### 2. `itemcf_results.csv`

保存不同 ItemCF 参数组合下的实验结果，包括：

* `k`
* `sim_metric`
* `min_common`
* 验证集 MAE / RMSE
* 测试集 MAE / RMSE

### 3. `mf_results.csv`

保存不同 BiasMF 参数组合下的实验结果，包括：

* `n_factors`
* `lr`
* `reg`
* `epochs`
* 验证集 MAE / RMSE
* 测试集 MAE / RMSE

### 4. `summary.json`

保存本次实验摘要信息，包括：

* 输入评分文件路径
* 训练集、验证集、测试集大小
* 最优基线模型
* 最优协同过滤模型
* 最优矩阵分解模型

## 八、推荐实验顺序

建议按以下顺序推进：

### 第一阶段：先跑通完整流程

先确认以下模型都能正常输出结果：

* GlobalMean
* UserMean
* ItemMean
* ItemCF
* BiasMF

### 第二阶段：做小规模参数实验

先做较少参数组合，快速观察趋势：

* ItemCF：`k = 10, 20`
* BiasMF：`n_factors = 8, 16`

### 第三阶段：做系统参数分析

逐步扩展参数范围，例如：

#### ItemCF

* `k = [5, 10, 20, 40, 80]`
* `sim_metric = [cosine, pearson]`

#### BiasMF

* `n_factors = [8, 16, 32, 64]`
* `lr = [0.001, 0.005, 0.01]`
* `reg = [0.005, 0.01, 0.05]`
