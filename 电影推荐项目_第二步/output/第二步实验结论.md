# 第二步实验结论

## 一、实验目标

本阶段主要完成经典推荐算法与图神经网络模型的学习、实现与实验对比，重点比较不同类型模型在评分预测任务中的表现，并分析模型参数对结果的影响。

## 二、实验设置

- 训练集大小：`80024`
- 验证集大小：`9988`
- 测试集大小：`9988`
- 主评价指标：平均绝对误差（MAE）、均方根误差（RMSE）
- 辅助评价指标：整数评分准确率、±0.5 / ±1.0 分容忍准确率、高分喜欢分类准确率、Precision、Recall、F1
- 对比模型：全局平均分、用户平均分、电影平均分、基于物品的协同过滤、带偏置矩阵分解、SVD++、图卷积网络、邻居聚合图网络

## 三、最优实验结果

### 1. 最优基线模型

- 模型名称：`电影平均分`
- 验证集平均绝对误差：`0.81337`
- 验证集均方根误差：`1.020164`
- 测试集平均绝对误差：`0.805682`
- 测试集均方根误差：`1.014071`
- 整数评分准确率：`0.376552`
- ±0.5 分容忍准确率：`0.380356`
- ±1.0 分容忍准确率：`0.675911`
- 高分喜欢分类准确率：`0.548358`
- 高分喜欢分类 Precision / Recall / F1：`0.790986` / `0.253247` / `0.383659`

### 2. 最优协同过滤模型

- 最优协同过滤参数为：邻居数 `40`，相似度方法为“皮尔逊相关系数”，最少共同评分数为 `2`。
- 测试集平均绝对误差为 `0.72214`，测试集均方根误差为 `0.922171`。
- 整数评分准确率：`0.431918`
- ±0.5 分容忍准确率：`0.431918`
- ±1.0 分容忍准确率：`0.736884`
- 高分喜欢分类准确率：`0.623949`
- 高分喜欢分类 Precision / Recall / F1：`0.832837` / `0.403499` / `0.543621`
- 说明基于相似物品的局部邻域信息能够有效提升评分预测精度。

### 3. 最优矩阵分解模型

- 最优矩阵分解参数为：隐向量维度 `8`，学习率 `0.005`，正则化系数 `0.01`，训练轮数 `20`。
- 测试集平均绝对误差为 `0.734968`，测试集均方根误差为 `0.930994`。
- Early stopping 选出的最佳轮数为 `20`。
- 整数评分准确率：`0.415699`
- ±0.5 分容忍准确率：`0.415699`
- ±1.0 分容忍准确率：`0.725471`
- 高分喜欢分类准确率：`0.602623`
- 高分喜欢分类 Precision / Recall / F1：`0.82881` / `0.358045` / `0.500063`
- 说明潜因子方法能够较好地挖掘用户与电影之间的隐含偏好关系。

### 4. 最优SVD++模型

- 最优SVD++参数为：隐向量维度 `8`，学习率 `0.005`，正则化系数 `0.01`，训练轮数 `20`。
- 测试集平均绝对误差为 `0.718736`，测试集均方根误差为 `0.912805`。
- Early stopping 选出的最佳轮数为 `20`。
- 整数评分准确率：`0.428314`
- ±0.5 分容忍准确率：`0.428314`
- ±1.0 分容忍准确率：`0.740589`
- 高分喜欢分类准确率：`0.620144`
- 高分喜欢分类 Precision / Recall / F1：`0.843676` / `0.387446` / `0.531026`
- 说明SVD++在潜因子建模基础上进一步引入了用户隐式反馈信息，能够更充分地利用用户历史交互信号。

### 5. 最优图神经网络模型

- 最优图神经网络模型为“图卷积网络”。
- 最优参数为：隐藏维度 `64`，图卷积层数 `1`，学习率 `0.003`，权重衰减 `5e-05`，训练轮数 `40`。
- 测试集平均绝对误差为 `0.732645`，测试集均方根误差为 `0.928802`。
- 说明在引入用户与电影身份表示、节点属性特征和评分边权后，图神经网络模型能够有效利用用户—电影图结构信息。

## 四、结果分析

- 在当前实验中，最优基线模型为“电影平均分”，测试集 RMSE 为 `1.014071`。
- 最优协同过滤模型测试集 RMSE 为 `0.922171`，相较最优基线下降约 `9.06%`。
- 最优矩阵分解模型测试集 RMSE 为 `0.930994`，相较最优基线下降约 `8.19%`。
- 最优SVD++模型测试集 RMSE 为 `0.912805`，相较最优基线下降约 `9.99%`。
- 最优图神经网络模型测试集 RMSE 为 `0.928802`，相较最优基线下降约 `8.41%`。
- 除主指标 MAE / RMSE 外，实验还补充了整数评分准确率、容忍误差准确率和高分喜欢分类指标，用于增强结果的可解释性。
- 按测试集 RMSE 从优到劣排序为：SVD++（0.912805），协同过滤（0.922171），图神经网络（0.928802），带偏置矩阵分解（0.930994），基线方法（1.014071）。

## 五、指标解释

- MAE 越小越好，表示平均每条评分预测偏差越小。
- RMSE 越小越好，且对大误差更敏感，是评分预测任务中的核心指标之一。
- exact_acc 越大越好，表示预测分四舍五入后与真实整数评分完全一致的比例。
- within_0_5_acc 和 within_1_0_acc 越大越好，表示预测值落在可接受误差范围内的比例。
- like_acc、precision、recall 和 f1 反映模型是否能识别用户真正喜欢的高分电影。

## 六、参数分析结论

### 1. 协同过滤参数分析

- 邻居数会直接影响预测时可利用的邻域信息数量。
- 邻居数过小会导致信息利用不足，邻居数过大则可能引入噪声。
- 不同相似度计算方法在当前数据集上的表现存在差异，应结合实验结果选择合适配置。

### 2. 带偏置矩阵分解参数分析

- 隐向量维度决定模型表示用户和电影潜在特征的能力。
- 学习率影响参数更新速度，过大可能不稳定，过小则收敛较慢。
- 正则化系数能够缓解过拟合。
- 训练轮数影响收敛程度，需要在训练充分与过拟合之间平衡。

### 3. SVD++参数分析

- SVD++ 在矩阵分解基础上引入了隐式反馈，因此通常比普通矩阵分解拥有更强的表达能力。
- 隐向量维度过小会限制表达能力，过大则可能增加训练成本并带来过拟合风险。
- 学习率与正则化系数共同影响训练稳定性和泛化能力，应通过验证集联合选择。
- 当用户交互历史较丰富时，SVD++ 往往更容易体现优势。

### 4. 图神经网络参数分析

- 隐藏维度会影响节点表示能力，维度过小可能表达不足，维度过大则可能增加训练难度。
- 图卷积层数过深时，容易出现节点表示过于平滑的问题，因此应控制传播层数。
- 学习率与权重衰减会共同影响训练稳定性和模型泛化能力。
- 在当前任务中，引入用户与电影身份表示、节点属性特征以及评分边权，是图模型取得较好效果的关键。

## 七、阶段结论

- 本阶段已经完成多种不同类型推荐算法的实现与对比，满足课程第二步要求。
- 从当前结果看，协同过滤、矩阵分解、SVD++ 和图模型均明显优于简单基线方法。
- 评分预测任务仍应以 MAE 和 RMSE 作为主指标。
- 准确率类指标和高分喜欢分类指标能够作为补充指标，增强实验结果的直观性和可解释性。
- SVD++ 在矩阵分解基础上进一步利用隐式反馈信息，为后续讨论显式反馈与隐式反馈融合提供了实验依据。
- 图神经网络在当前数据集上的表现已经接近或达到较优水平，说明图结构建模在显式评分预测任务中具有较强潜力。
- 后续可继续从不同用户群体、不同电影群体和不同评价指标角度，分析模型在不同层次上的表现差异。
