# 电影推荐项目第一步：预处理与可视化分析

本项目对应课程要求中的第一阶段任务：
- 读取电影评分、用户、电影元数据
- 完成基础预处理
- 输出关键统计指标
- 绘制可视化图像
- 自动生成阶段性分析结论

## 一、建议目录

请将原始数据集放到如下目录：

```text
data/
└── ml-100k/
    ├── u.data
    ├── u.item
    ├── u.user
    ├── u.genre
    ├── u.occupation
    └── 其他原始文件
```

## 二、安装依赖

```bash
pip install -r requirements.txt
```

## 三、运行方式

```bash
python run_step1.py --data-dir data/ml-100k --output-dir output
```

如果你的数据放在别的位置，可以把 `--data-dir` 改成你的真实路径。

## 四、运行后输出内容

运行完成后会在输出目录生成：

```text
output/
├── 数据/
│   ├── 评分表_预处理后.csv
│   ├── 用户表_预处理后.csv
│   ├── 电影表_预处理后.csv
│   ├── 用户统计.csv
│   ├── 电影统计.csv
│   ├── 类型统计.csv
│   └── 全局统计.json
├── 图像/
│   ├── 评分分布.png
│   ├── 用户活跃度分布.png
│   ├── 电影热度分布.png
│   ├── 用户年龄分布.png
│   ├── 用户性别分布.png
│   ├── 电影类型统计.png
│   └── 热门电影前十.png
└── 阶段一分析结论.md
```

## 五、这一阶段可以回答的问题

1. 数据有多稀疏
2. 用户评分是否均衡
3. 电影热度是否具有长尾特征
4. 用户年龄和性别分布如何
5. 电影类型分布如何
6. 哪些现象会影响后续推荐算法效果

## 六、建议你后续继续做的内容

1. 先做一个基线方法：全局平均分、用户平均分、电影平均分
2. 再做协同过滤：用户协同过滤或物品协同过滤
3. 再做矩阵分解：如基础矩阵分解或带偏置的矩阵分解
4. 最后做参数分析和分群分析
