# 电影推荐项目第一步：用户和电影数据的统计分析与可视化

本项目对应课程任务中的第一阶段，目标是对 MovieLens 100K 数据集进行基础清洗、统计分析和可视化，为后续协同过滤、矩阵分解和图模型实验提供数据基础。

当前阶段主要完成以下内容：

- 读取评分数据、用户数据、电影元数据
- 完成基础预处理与异常清洗
- 构建用户统计、电影统计、类型统计等结果表
- 输出全局统计指标
- 绘制多种可视化图像
- 分析数据稀疏性、长尾分布和冷启动现象对后续推荐建模的影响

## 七、这一阶段重点回答的问题

通过本阶段的统计分析与可视化，可以回答以下问题：

1. 数据集一共有多少用户、多少电影、多少评分
2. 评分矩阵是否高度稀疏
3. 用户行为是否呈现明显的活跃度差异
4. 电影热度是否具有长尾分布特征
5. 冷启动用户和冷启动电影比例有多高
6. 用户年龄、性别、职业分布如何
7. 电影类型和上映年代分布如何
8. 这些数据特征会怎样影响后续推荐算法效果

## 八、这一阶段对后续建模的意义

第一步并不只是“做统计图”，更重要的是为后续推荐算法提供依据。

### 1. 稀疏性影响协同过滤效果

如果评分矩阵非常稀疏，用户之间或电影之间的相似度计算会不稳定，协同过滤效果容易下降。

### 2. 长尾分布会带来热门偏置

如果少量热门电影获得了大量评分，模型容易偏向热门电影，导致冷门电影曝光不足。

### 3. 冷启动问题会影响新用户和新电影

评分次数很少的用户或电影，很难仅依靠历史交互做出准确推荐，因此后续可能需要引入内容特征、偏置项或图结构信息。

### 4. 用户分层有助于后续实验分析

把用户划分为低活跃、中活跃、高活跃后，可以在后续实验中分别分析不同用户群体上的模型效果，而不是只看总体平均指标。
